Views control how you see and interact with your data. Twenty gives you multiple view types, powerful filtering, and the ability to save custom views for different workflows.

## View types

<CardGroup cols={3}>
  <Card title="Table View" icon="table">
    Spreadsheet-style rows and columns. Sort, group, resize, and reorder columns. Best for working with many records at once.
  </Card>
  <Card title="Kanban View" icon="columns-3">
    Drag-and-drop cards organized by a select field (like deal stage). Perfect for pipelines and project tracking.
  </Card>
  <Card title="Calendar View" icon="calendar">
    Records plotted on a calendar by a date field. Ideal for tracking tasks, meetings, or deadlines.
  </Card>
</CardGroup>

## Filters & sorting

Filter your data with conditions like equals, contains, greater than, is empty, and more. Combine filters with AND/OR logic for complex queries. Sort by multiple fields and group records by any select or relation field.

## Saved views

Create custom views with specific filters, sorts, columns, and view types — then save them for your team. Control who can see and edit each view with view-level permissions.

## Pipelines

A pipeline is a kanban view configured for deal tracking. Set up stages (like Qualification, Proposal, Negotiation, Closed Won) and drag opportunities between them. Add expected amounts and track time-in-stage.
